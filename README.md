# Playwright with Python
Repository to learn and improve testing using Playwright

## Config
Before starting you will need an environment file `.env` to locate secrets or environment variables. For example:
```
OPENCODE_API_KEY=<YOUR_OPENCODE_API_KEY>
OPENCODE_DEFAULT_MODE=opencode/kimi-k2.5-free
GITHUB_TOKEN=<YOUR_TOKEN>
OPENAI_API_KEY=<YOUR_OPENAI_API_KEY>
```
